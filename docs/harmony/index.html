当前版本范围

  • 支持本地目标管理、本地计划查看、本地流水记录。
  • 支持本地 JSON 导入与导出。
  • 支持每日提醒与交互震动反馈。
  • 当前版本不提供内购、会员订阅、云同步、iCloud 与小组件能力。
